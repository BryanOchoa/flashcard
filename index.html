Top 15 HTML Tags

Created by:

Tag Description
<!DOCTYPE> Defines the document type and HTML version.
<html> The root element that contains all content on the page.
<head> Contains meta-information and links to external resources.
<title> Sets the title of the webpage displayed in the browser tab.
<meta> Provides metadata about the document.
<link> Links to external resources like stylesheets.
<script> Embeds or references JavaScript code.
<style> Contains CSS rules for styling the HTML document.
<body> Contains the visible content of the webpage.
<h1>, <h2>, ... <h6> Headings for structuring content.
<p> Defines a paragraph of text.
<a> Creates hyperlinks to other web pages or resources.
<img> Embeds images in the webpage.
<ul> Defines an unordered (bulleted) list.
<ol> Defines an ordered (numbered) list.
